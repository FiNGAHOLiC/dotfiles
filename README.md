# dotfiles

My config files :)

## Usage

```
$ cd ~
$ git clone git@github.com:FiNGAHOLiC/dotfiles.git
$ ln -s dotfiles/.vimrc .vimrc
$ ln -s dotfiles/.gvimrc .gvimrc
$ ln -s dotfiles/.vim .vim
$ cd dotfiles
$ git submodule init
$ git submodule update
```

Then, launch `vim` and run `:NeoBundleInstall`.
